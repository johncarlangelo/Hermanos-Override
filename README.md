# Sandbox Unlocked (⚠️ WORK IN PROGRESS ⚠️)

**Sandbox Unlocked** is a customizable, offline trainer launcher designed for single-player games. It provides a clean interface for toggling cheats, editing in-game values, and unlocking sandbox-style gameplay — all without relying on external platforms.

---

## 🎮 Features (Planned / In Progress)

- ✅ Load custom trainer files (JSON or script-based)
- ✅ Attach to any running single-player game process
- ✅ Memory editing: set or freeze values (e.g., health, gold)
- ⏳ UI with toggles, sliders, and hotkeys
- ⏳ Profile-based trainer layout per game
- ⏳ Lightweight and portable — no installer needed

---

## 🚀 Goal

To build a sleek and personal trainer system that gives full control to the player — no restrictions, no bloat, just sandbox freedom after beating a game.

---

## ⚙️ Technologies Used

- Cheat Engine (for address discovery)
- Python / C#
